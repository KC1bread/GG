# 相对论航行者 — 使用说明

## 这是什么

一个沉浸式相对论 3D 可视化交互系统。驾驶飞船以接近光速穿越太阳系，**亲眼看到**时间膨胀、长度收缩、光行差等相对论效应。

## 快速上手

1. 点击 **「开始任务」** 进入
2. 长按 **Shift** 加速，按 **Ctrl** 减速
3. **WASD** 飞行，**鼠标右键拖拽** 环顾四周
4. 观察右侧 HUD 面板中 γ 值和时间的变化

## 右侧控制面板

### 速度 β = v/c

拖动滑块或用 Shift/Ctrl 调节。预设按钮可快速跳转到 0.1c、0.5c、0.8c、0.9c、0.99c。

### 参考系

| 模式 | 含义 |
|------|------|
| 地球参考系 | 站在地球立场看飞船 |
| 飞船参考系 | 站在飞船立场看宇宙 |
| 并列比较 | 左右分屏同时对比 |

### 视角

| 模式 | 含义 |
|------|------|
| 第三人称 | 从飞船后方观看 |
| 第一人称 | 座舱视角，有仪表盘和 HUD 准星 |

### 显示模式

| 模式 | 含义 |
|------|------|
| **Measured** | 物理测量值（尺子量出来的长度、时钟走过的时间） |
| **Observed** | 视觉观察效果（眼睛实际看到的样子，包含光行差和 Terrell 旋转） |

### Terrell 效果（仅 Observed 模式）

| 模式 | 效果 |
|------|------|
| 纯 Lorentz 收缩 | 物体沿飞行方向压扁 |
| Penrose-Terrell 精确 | 收缩 + 旋转，高速下球体看起来仍是圆的 |
| 增强教学 | 旋转放大 1.5 倍，更容易观察 |

## 各面板简介

### Relativity HUD
显示速度 β、洛伦兹因子 γ、地球时间、飞船固有时间、距离、到达 ETA、长度收缩比例，以及地球钟 vs 飞船钟的双钟对比。

### 双测量尺（右下角）
3D 小窗显示平行于和垂直于飞行方向的测量尺。在 Observed 模式下平行尺会产生 Terrell 旋转。可拖拽旋转视角。

### Minkowski 时空图
纵轴是时间 ct，横轴是空间 x。飞船越快，世界线越贴近光锥（虚线）。

### 学习任务
5 个引导式任务，从低速直觉到高速相对论效应，逐步深入。

### 概念解释
四个层级的概念卡片：现象 → 直观解释 → 公式 → 参考系。

### 概念问答
4 道选择题，测试对时间膨胀、长度收缩、参考系等概念的理解。

### 实验记录
自动记录所有操作（调速度、切模式、答题目等），可导出 JSON 或 CSV。

## 值得一看的效果

1. **把速度拉到 0.9c**，看 HUD 中地球钟和飞船钟的差距逐渐拉大
2. **切换到 Observed 模式**，观察星光汇聚前方、颜色偏移（光行差）
3. **切换 Terrell 模式对比**：纯收缩时行星压扁，精确模式下行星看起来仍是圆的
4. **打开并列比较**，看地球系和飞船系中测量尺的不同
5. **按 P 进入自由视角**，拖拽观察飞船外部的 Terrell 旋转随视角变化
