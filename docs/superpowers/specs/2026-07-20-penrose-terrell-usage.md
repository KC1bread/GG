# Penrose-Terrell 效果 — 使用说明

## 是什么

高速飞行时，物体会因为相对论效应发生**视觉旋转**。即使物体本身是球形，在接近光速时从任何角度看它都仍然是圆的 — 这不是因为它没变，而是 Lorentz 收缩和光行差共同作用的结果，被称为 Penrose-Terrell 旋转。

## 怎么用

### 进入 Observed 模式

右侧面板顶部选择 **Observed（观察者视角）**，Terrell 模式选择器会自动出现。

### 三种模式

| 模式 | 效果 |
|------|------|
| **纯 Lorentz 收缩** | 物体只沿飞行方向压扁，不旋转 |
| **Penrose-Terrell 精确**（默认） | 物理正确的收缩 + 旋转 |
| **增强教学** | 旋转放大 1.5 倍，更容易观察到 |

### 加速飞行

- 长按 **Shift** 加速到 β ≈ 0.9（接近光速）
- 按 **Ctrl** 减速
- 速度越快，效果越明显

### 观察什么

1. **行星** — 高速飞过时，它们看起来仍然是圆的（精确模式），但纯收缩模式下会压扁
2. **飞船（第三人称）** — 拖拽视角，飞船的旋转角度随你的观察方向变化
3. **右下角测量尺** — 平行尺会随视角产生俯仰旋转
